# Contributing

GloGEM is developed and maintained by the VAW glaciology group at ETH Zürich. The model code and this documentation live together in one repository and accept contributions from group members:

https://github.com/eth-vaw-glaciology/GloGEM

The repository uses a shared `main` branch. All contributors push directly to `main`, so coordinating changes and keeping your local copy up to date is important to avoid conflicts.
